# League of Legends API Website

## Overview

This project is a web application that connects to an external API to retrieve and display data related to League of Legends. The goal of the website is to present api data into a javascript and node.js powered website.

---

##  What This Website Does

* Sends requests to a League of Legends API
* Receives structured data in JSON
* Processes and formats that data
* Displays it on a webpage for users to view

---

## What I Learned

* How to access an API using node.js architecture
* How to handle API responses returned in JSON format
* How to extract and use data from those responses
* How to display dynamic data on a website
* How the API and the server communicate

---

## How It Works (Simple Explanation)

1. The website makes a request to an API endpoint
2. The API returns data
3. The website processes that data
4. The data is displayed on the page

## Technologies Used

* HTML / CSS
* JavaScript
* Node.js
* External League of Legends API
